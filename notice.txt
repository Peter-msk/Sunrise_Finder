# Sunrise Finder v2.0.0 업데이트 안내

Sunrise Finder는 주소와 공간정보를 중심으로 여러 공공데이터를 연결하여 실무자가 필요한 정보를 보다 빠르게 검색하고 검토할 수 있도록 개발된 업무 지원 프로그램입니다.

이번 버전은 도로명주소와 지번주소를 기준으로 대상 위치를 찾고, 관련 대지와 건축물 정보를 확인하며, VWorld 지도와 공간정보 레이어를 이용해 주변 현황을 검토할 수 있도록 구성했습니다.

## 주요 활용 목적

* 도로명주소·지번주소 검색 및 위치 확인
* 하나의 주소와 연관된 관련 대지 검토
* 건축물대장과 건축 관련 공공정보 조회
* VWorld 배경지도 및 공간정보 레이어 검색
* 사업지구, 조사지점 및 분석범위 확인
* 환경영향평가와 GIS 실무를 위한 기초자료 정리
* 반복적인 주소·건축물·공간정보 확인 업무의 간소화

## 이번 배포본의 방향

이번 배포본은 여러 조회 기능을 하나의 프로그램에서 사용할 수 있도록 통합하고, 사용자가 검색 과정과 처리 상태를 확인할 수 있도록 구성하는 데 중점을 두었습니다.

프로그램은 원본 자료를 직접 변경하지 않으며, 조회 결과와 사용자 작업정보를 분리하여 관리합니다. 외부 API를 사용하는 기능은 제공기관의 응답 상태와 자료 등록 여부에 따라 결과가 달라질 수 있습니다.

건축물 정보가 조회되지 않는 경우 반드시 프로그램 오류를 의미하는 것은 아닙니다. 해당 주소에 자료가 등록되지 않았거나, 도로명주소와 지번주소의 연결 상태, 관련 대지 구성, 공공데이터 제공 범위 또는 API 이용 상태에 따라 무자료 결과가 발생할 수 있습니다.

## 사용 시 유의사항

Sunrise Finder는 공식 증명서 발급 프로그램이 아니며, 행정기관의 최종 판단이나 법적 검토를 대신하지 않습니다. 중요한 업무에 사용하는 자료는 반드시 해당 공공기관의 공식 시스템과 원문자료를 통해 다시 확인해야 합니다.

현재 버전은 비공개 베타테스트를 위한 실행형 프로그램입니다. 기능 검증이 완료되기 전까지 외부 공개, 무단 복제 및 재배포를 금지합니다.

향후에는 베타테스트 결과를 반영하여 조회 안정성, 지도 표시 성능, 공간정보 검색, 건축물 정보 구성, 오류 안내와 사용자 편의성을 지속적으로 개선할 예정입니다.
