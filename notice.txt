Sunrise Finder v2.1.0 개인용 정식 버전입니다.

- 현장사진 작업 버튼을 실무 순서에 따라 한 줄로 재배치했습니다.
- 현장사진 목록은 파일명, 연결지점, 경도, 위도, 고도, 촬영방향, 경로지도, 공간지도, 상태 9열을 유지합니다.
- 사진 마커 팝업과 건축HUB 주소 공백 정규화 보완을 유지합니다.
- 지도 스냅샷은 흰 WebEngine 프레임을 저장하지 않고 실제 렌더링을 재시도합니다.
- Sunrise Group Authenticode 서명과 ZIP SHA256 검증을 필수로 수행합니다.
- 승인된 개인 API 키는 평문 파일 없이 Base85 payload와 런타임 keystream 방식으로 난독화되어 포함됩니다.
- 난독화는 복구 가능하므로 이 패키지와 업데이트 파일은 공개 저장소에 배포하면 안 됩니다.
- 업데이트 자리표시자 `.invalid` 호스트를 차단하고, 실제 개인용 배포 주소가 없으면 자동 다운로드를 비활성화합니다.
- 업데이트 연결 오류의 기술 예외 원문 대신 사용자용 한국어 안내를 표시합니다.
